# 📊 Customer Data Analysis using SQL

## 🚀 Project Overview  
This project focuses on analyzing customer data using **SQL queries**.  
The goal is to extract meaningful insights from raw data stored in a CSV file by using structured queries.

---

## 📁 Project Files  

### 1️⃣ Dataset File (.csv)  
Contains customer-related data such as:
- Customer ID  
- Age  
- Gender  
- Income  
- Spending Score  
- Purchase History  
- Product Category  

This file serves as the **raw data source** for analysis.

---

### 2️⃣ SQL File (.sql)  
Contains all SQL queries used in the project, including:
- Table creation  
- Data insertion  
- Data filtering  
- Aggregations (SUM, AVG, COUNT)  
- Grouping (GROUP BY)  
- Sorting (ORDER BY)  
- Conditional queries (WHERE, HAVING)  
- Advanced queries (Subqueries, Joins, Window Functions)  

---

## 🛠️ Tech Stack  
- SQL (MySQL / PostgreSQL)  
- CSV Dataset  

---

## 🔍 Project Workflow  

1. Load CSV data into database  
2. Create table using SQL  
3. Insert or import data  
4. Perform analysis using SQL queries  
5. Generate insights from query results  

---

## 📈 Key Analysis Performed  
- Customer segmentation based on spending  
- Income vs spending analysis  
- Category-wise purchase insights  
- Identification of high-value customers  
- Customer behavior patterns  

---

## ▶️ How to Use  

### Step 1: Import Dataset  
- Import the `.csv` file into your database  

### Step 2: Run SQL File  
- Open `.sql` file in MySQL/PostgreSQL  
- Execute queries step-by-step
